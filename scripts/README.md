Scripts
=======

This directory will contain helper scripts for development, testing, and maintenance.

Examples:

- `setup_dev_env.*` – Install dependencies and set up local development
- `start_dev.*` – Start frontend, backend, and ML services for local development
- `collect_cctv_samples.*` – Capture and organize CCTV frames for training
- `backup_db.*` – Backup and restore database data
