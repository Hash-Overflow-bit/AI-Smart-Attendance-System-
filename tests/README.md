Tests
=====

This directory will contain cross-cutting tests for the system, separate from the backend and frontend local tests.

Suggested substructure:

- `e2e/` – End-to-end tests covering full user flows (e.g., login, attendance capture, analytics)
- `load_tests/` – Performance and load tests
- `security_tests/` – Security and permission tests
