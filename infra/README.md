Infrastructure
==============

This directory will contain deployment and environment configuration for the smart attendance system.

Suggested substructure:

- `docker/` – Dockerfiles and related configuration for frontend, backend, and ML services
- `k8s/` – Kubernetes manifests (if used)
- `ci/` – Continuous integration configuration (e.g., GitHub Actions, Azure Pipelines)
- `monitoring/` – Monitoring and observability configuration
